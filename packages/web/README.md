# @tesseron/web

Browser SDK for [Tesseron](../../README.md).

Declares actions and subscribable resources in your web app and opens a WebSocket to the local Tesseron gateway. Exports `WebTesseronClient`, `BrowserWebSocketTransport`, the default `tesseron` singleton, and `DEFAULT_GATEWAY_URL` (`ws://localhost:7475`); re-exports the `@tesseron/core` builder and context types.
