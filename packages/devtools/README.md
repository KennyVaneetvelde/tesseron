# @tesseron/devtools

In-browser debug UI for [Tesseron](../../README.md). Served by the MCP gateway at `http://localhost:7475/__devtools` when in dev mode.

Live invocations, timing, sample/elicit traces, error inspector. Lazy-loaded; not bundled into production builds.

## Status

Scaffold (Phase 4 in the [PRD](../../.planning/requirements.md)).
